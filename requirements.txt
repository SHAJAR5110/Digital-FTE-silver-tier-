# First Time Setup

1. Install Python dependencies:
   pip install -r requirements.txt

2. Setup Google OAuth for Gmail:
   - Go to https://console.cloud.google.com/
   - Create new project
   - Enable Gmail API
   - Create OAuth 2.0 credentials (Desktop app)
   - Download credentials.json
   - Save to ~/.secrets/gmail_credentials.json
   - Add to .env: GMAIL_CREDENTIALS_PATH=~/.secrets/gmail_credentials.json

3. Setup WhatsApp access:
   - Option A: Use Playwright with web.whatsapp.com (free)
   - Option B: Use official WhatsApp Business API (paid)

4. Setup LinkedIn:
   - Option A: Use unofficial API (linkedin-api)
   - Option B: Use Playwright web scraping
   - Option C: Official LinkedIn API (requires business account)

5. Setup Email Sending:
   - Option A: Gmail API (recommended)
   - Option B: SMTP (any email provider)
   - Get credentials and add to .env

# Environment Variables (.env file)

```
# Gmail
GMAIL_CREDENTIALS_PATH=~/.secrets/gmail_credentials.json

# Email Sending
EMAIL_PROVIDER=gmail  # OR smtp
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_EMAIL=your_email@gmail.com
SMTP_PASSWORD=your_app_password

# LinkedIn (if using unofficial API)
LINKEDIN_EMAIL=your_email@example.com
LINKEDIN_PASSWORD=your_password

# WhatsApp (if using unofficial API)
WHATSAPP_SESSION_PATH=~/.secrets/whatsapp_session

# Vault
VAULT_PATH=/path/to/AI_Employee_Vault

# Scheduling
CLAUDE_RUN_INTERVAL=1800  # 30 minutes in seconds
```

NEVER commit .env file to git!

# Troubleshooting

## Gmail Auth Error
- Verify credentials.json in correct path
- Check permissions in Google Cloud Console
- Re-authenticate if needed

## Playwright Timeout
- Chrome might be slow on first run
- Increase timeout in whatsapp_watcher.py
- Ensure 2GB+ RAM available

## LinkedIn Login Failed
- Session might be expired
- Clear session files and re-login
- Check if account has 2FA enabled

## Email Send Failed
- Verify SMTP credentials
- Check firewall allowing port 587
- Less secure apps might need enabling (Gmail)
